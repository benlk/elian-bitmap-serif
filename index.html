Download .ttf

To use this font, first download these font files and save them to your server. Then use a @font-face declaration in your CSS to load the files from your server. 


<style type="text/css">
	@font-face {
		font-family: "Elian Bitmap Serif"
		src: url('./Elian Bitmap Serif.ttf')
		;
		font-weight: normal;
	}
	.ebs {
		font-family: "Elian Bitmap Serif", sans-serif;
	}
</style>

This font can also be used on computers that support TrueType fonts.
